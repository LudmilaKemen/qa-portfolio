# QA Portfolio

## Описание
Проект содержит тестовые задания, выполненные в рамках практики QA-инженера.

## Стек:
- Python
- Postman
- Git
- JSON
- SQL (базовый уровень)

## Выполненные задачи:

### Task 1 — алгоритмы
Реализация обхода кругового массива.

### Task 2 — геометрия
Определение положения точек относительно эллипса.

### Task 3 — работа с API-структурами
Объединение tests.json и values.json в report.json.

### Task 4 — оптимизация
Поиск минимального количества операций для выравнивания массива.

## API Testing
Проводилось тестирование REST API с использованием Postman:
- проверка status codes
- проверка JSON структуры
- валидация бизнес-логики

## Цель проекта
Практика навыков QA-инженера и подготовка к стажировке.
Реализована логика обработки API-ответов:
- объединение структуры тестов
- сопоставление результатов по id
- генерация итогового отчета
## What I would improve

- Add more negative test cases for API testing
- Extend test coverage for edge cases
- Add bug reports for simulated defects
- Improve documentation of test strategy

The tasks were implemented with focus on:
- input validation
- edge cases handling
- data consistency
- API response verification
- reproducibility of test results
